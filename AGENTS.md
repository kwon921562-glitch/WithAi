# 프로젝트 에이전트 지침

이 저장소에서 작업할 때:

- 기본 언어는 Lua 이다.
- 모듈은 작게 유지하고 암묵적 전역을 피한다.
- Love2D 코드에서는 update 와 draw 책임을 분리한다.
- lua_tuto/ 의 기존 튜토리얼 순서와 파일 이름을 유지한다.
- 수정 시 변경 파일과 검증 실행 방법을 함께 설명한다.

## 검증

- Lua 파일 실행: task "Lua: Run current file"
- Love2D 앱 실행: task "Love2D: Run project"
- 프로젝트 도구가 있으면 포맷/린트 규칙 일관성을 유지한다
