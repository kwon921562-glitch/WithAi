# 팀 AI 컨벤션 (도구 중립)

## 핵심 원칙

- 전역보다 로컬 모듈을 우선한다.
- 함수는 가능한 작고 순수하게 유지한다.
- 외부 입력은 항상 nil 가능성을 처리한다.
- Love2D 프레임 업데이트 로직은 결정적으로 유지한다.

## AI 어시스턴트 출력 규칙

- 변경 파일을 먼저 제시한다.
- 동작 변화와 리스크를 함께 설명한다.
- 최소 검증 체크리스트를 포함한다.

## Lua/Love2D 코딩 표준

- Love2D 프로젝트에서는 LuaJIT/Lua 5.1 호환 문법을 사용한다.
- 암묵적 전역 변수에 의존하지 않는다.
- update 와 draw 책임을 명확히 분리한다.
- 유틸리티 모듈에는 부수 효과를 넣지 않는다.
